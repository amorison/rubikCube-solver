rubikCube-solver
================

A graphical Rubik's cube solver made with Python and Pygame.

Commands :
- 8 : rotate UP layer
- 5 : rotate FACE layer
- 2 : rotate DOWN layer
- 0 : rotate BACK layer
- 4 : rotate LEFT layer
- 6 : rotate RIGHT layer
- b : scramBle the cube randomly
- s : Solve the cube (using a naive method, easier to implement :p )
